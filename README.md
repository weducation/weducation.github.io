# weducation.github.io

Website para testes.

<https://weducation.github.io>

## Como subir no modo desensolvimento
1. Instale os pacotes dependentes:

`npm i`

2. Suba o servidor de desenvolvimento:

`npm run dev`

3. Acesse:

<http://localhost:8080/>

## Como fazer o deploy

1. Execute o comando de build:

`npm run build`

2. Adicione tudo no commit:

`git add .`

3. Insira um resumo das alterações:

`git commit -am 'Resumo das alterações aqui'`

4. Faça o push para o GitHub

`git push`
